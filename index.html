Auto Import — How It Works
Your bank sends you an SMS for every transaction → your phone forwards it → FinanceOS imports it automatically (internal transfers between your own accounts are auto-detected and excluded from income/expense)

Step 1 — Firebase Setup
Required — your SMS messages are stored here
Connected
Step 2 — Create SMS Collection in Firestore
One-time setup — run this SQL in Firebase Console
In Firebase Console → Firestore → Rules — add this collection rule:
match /financeos_sms/{document} {
  allow read, write: if true;
}
This allows the SMS forwarder app to write incoming SMS to Firestore.
Your Firestore collection path: financeos_sms · Document auto-generated per SMS
Step 3 — Deploy Cloud Function Webhook
Creates a secure HTTPS URL for your iPhone Shortcut to POST to
📋 One-time deploy (on your laptop/PC):
1. Install Node.js → nodejs.org
2. npm install -g firebase-tools
3. firebase login
4. Download the function files from this app (index.js, package.json, firebase.json)
5. cd functions && npm install
6. firebase use financeos-vamsi
7. firebase deploy --only functions
→ You get: https://us-central1-financeos-vamsi.cloudfunctions.net/smsWebhook
⚠️ Requires Blaze plan on Firebase (free quota = 2M req/month — personal use is always free).
Your secret auth token:
FinanceOS-Vamsi-2026
Change this in index.js line 5 before deploying. Never share it.
Step 3b — Create iOS Shortcut
New SMS received → POST directly to Firestore — no Cloud Function needed
POST this URL in your Shortcut (auto-filled from your Firebase config):
Configure Firebase above to see URL
A — Create the Shortcut
1. Shortcuts app → tap +
2. Add action: Receive Input (Text)
3. Add action: Get Contents of URL
  • URL: paste the URL above
  • Method: POST
  • Request Body: JSON
  • Add key fields → value:
    {"body":{"stringValue":"[Shortcut Input]"},"processed":{"booleanValue":false},"ts":{"stringValue":"[Current Date ISO8601]"}}
4. Name it "FinanceOS SMS" → Done
B — Create Automation
1. Shortcuts → Automation tab → +
2. Choose: Message
3. Sender: Any Contact (or your bank senders like VM-SBIINB, VD-HDFCBK)
4. Tap Next
5. Add action: Run Shortcut → "FinanceOS SMS"
6. Set Input: Message Content
7. Disable "Ask Before Running" ← critical
8. Done ✓
Exact JSON body to paste in the Shortcut's "Get Contents of URL" → Request Body:
{"fields":{"body":{"stringValue":"PASTE_SHORTCUT_INPUT_VARIABLE_HERE"},"processed":{"booleanValue":false},"ts":{"stringValue":"PASTE_CURRENT_DATE_ISO8601_VARIABLE_HERE"}}}
Replace the two PASTE_… placeholders with the actual Shortcut variables using the variable picker (tap the placeholder text → insert variable)
✅ Every bank SMS now lands in Firestore financeos_sms automatically. Pull below to preview and import.
