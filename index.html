Step 1 — Firebase Setup
Required — your SMS messages are stored here
Connected
Step 2 — Create SMS Collection in Firestore
One-time setup — run this SQL in Firebase Console
In Firebase Console → Firestore → Rules — add this collection rule:
match /financeos_sms/{document} {
  allow read, write: if true;
}
This allows the SMS forwarder app to write incoming SMS to Firestore.
Your Firestore collection path: financeos_sms · Document auto-generated per SMS
Step 3 — Set up on iPhone (iOS Shortcuts)
Use Apple's free Shortcuts app to forward bank notifications to Firebase
ℹ️ iPhone limitation: iOS does not allow any app to read your SMS messages directly. However, you can use Apple Shortcuts + Automation to capture bank notification text and send it to Firebase automatically.
✅ Option A — iOS Shortcuts Automation
1. Open Shortcuts app → tap Automation tab
2. Tap + → App → select your bank app (SBI YONO, HDFC, ICICI iMobile etc.)
3. Choose "When Opened" — or better: Notification trigger
4. Add action: Get Contents of URL
5. Set URL to your Firebase webhook (shown below)
6. Method: POST · Body: JSON with notification text
7. Turn off "Ask Before Running"
✅ Option B — Bank Email Alerts
Most banks send transaction emails too:
1. Log in to your bank app/website
2. Enable email alerts for all transactions
3. Use iOS Shortcuts → Email trigger on Gmail/Mail app
4. When email from bank arrives → forward to Firebase

Works reliably on iPhone ✓
iOS Shortcut — Exact steps
In Shortcuts app → Automation → New Automation:
• Trigger: Notification received from your bank app
• Action 1: Get Details of Notification → get "Body"
• Action 2: Get Contents of URL:
Firebase URL (copy this):
Configure Firebase first (Data Sync page)
• Method: POST · Content Type: JSON
• Body: {"fields":{"msg":{"stringValue":"<Shortcut Input>"}}}
• Replace <Shortcut Input> with the notification body variable
• Turn off "Ask Before Running"
💡 Most practical iPhone approach: Enable bank email alerts → copy the week's transaction emails → paste into Bulk SMS Import (it parses email text too). Takes 1 minute for a week of transactions. The iOS Shortcuts method works but requires initial setup and may need re-authorization after iOS updates.
