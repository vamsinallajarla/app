Step 1 — Firebase Setup
Required — your SMS messages are stored here
Connected
Step 2 — Create SMS Collection in Firestore
One-time setup — run this SQL in Firebase Console
In Firebase Console → Firestore → Rules — add this collection rule:
match /financeos_sms/{document} {
  allow read, write: if true;
}
This allows the SMS forwarder app to write incoming SMS to Firestore.
Your Firestore collection path: financeos_sms · Document auto-generated per SMS
Step 3 — Deploy Cloud Function Webhook
Creates a secure HTTPS URL for your iPhone Shortcut to POST to
📋 One-time deploy (on your laptop/PC):
1. Install Node.js → nodejs.org
2. npm install -g firebase-tools
3. firebase login
4. Download the function files from this app (index.js, package.json, firebase.json)
5. cd functions && npm install
6. firebase use financeos-vamsi
7. firebase deploy --only functions
→ You get: https://us-central1-financeos-vamsi.cloudfunctions.net/smsWebhook
⚠️ Requires Blaze plan on Firebase (free quota = 2M req/month — personal use is always free).
Your secret auth token:
FinanceOS-Vamsi-2026
Change this in index.js line 5 before deploying. Never share it.
Step 3b — Create iOS Shortcut
Captures bank notification text → sends to your Cloud Function
Create the Shortcut
1. Shortcuts app → tap +
2. Add: Receive Input from Shortcut
3. Add: Get Contents of URL
  • URL: your Cloud Function URL
  • Method: POST
  • Headers: Authorization: Bearer FinanceOS-Vamsi-2026
  • Body: JSON → rawMessage: Shortcut Input
4. Name: "Send to FinanceOS"
Create Automation
1. Automation tab → +
2. App → select your bank app
3. "Is Opened" → Next
4. Add: Run Shortcut → "Send to FinanceOS"
5. Input: Last Notification
6. Disable "Ask Before Running"
7. Done ✓
✅ Now when you open your bank app after a transaction, the notification is automatically sent to Firebase. Then pull below to import.
